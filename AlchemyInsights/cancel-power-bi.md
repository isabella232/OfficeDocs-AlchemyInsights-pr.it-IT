# <a name="cancel-power-bi"></a>Annulla Power BI

Power BI può essere annullato nella interfaccia di amministrazione di Microsoft 365.

1. Nell'interfaccia di amministrazione passare alla pagina **Fatturazione > [I tuoi](https://go.microsoft.com/fwlink/p/?linkid=842054)** prodotti.
2. Nella scheda **Prodotti** trovare l'abbonamento che si vuole cancellare, selezionare i tre puntini (Altre azioni), quindi selezionare **Annulla abbonamento**.
3. Nel riquadro **Annulla abbonamento** scegliere un motivo per cui si vuole annullare l'abbonamento. Se si vuole, inviare un feedback.
4. Selezionare **Salva**.

**Rimborso: qualsiasi credito rievato verrà restituito all'utente entro il ciclo di fatturazione successivo.**

**Nota:** se l'opzione di annullamento non è disponibile, selezionare l'abbonamento per visualizzare la pagina dei dettagli. Nella sezione **Impostazioni sottoscrizione e pagamento** selezionare Annulla **sottoscrizione.**

Per ulteriori informazioni, vedere [Annullare l'abbonamento.](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription)
