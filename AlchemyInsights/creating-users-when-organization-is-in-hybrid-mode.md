# <a name="create-users-in-hybrid-deployments"></a>Creare utenti nelle distribuzioni ibride

Quando l'organizzazione è una distribuzione ibrida tra un ambiente locale e Microsoft 365, è necessario creare tutti gli oggetti utente in Active Directory locale. Gli oggetti utente vengono creati nell'unità organizzativa appropriata in modo che siano sincronizzati nell'organizzazione.
