# <a name="help-signing-in-with-windows-hello-pin"></a>Assistenza nell'accesso tramite PIN di Windows Hello

- Per cambiare, reimpostare o ripristinare il PIN (una volta eseguito l'accesso), visitare [Cambia il PIN una volta eseguito l'accesso al dispositivo](https://support.microsoft.com/windows/change-your-pin-when-you-re-already-signed-in-to-your-device-0bd2ab85-b0df-c775-7aef-1324f2114b19).

- Per cambiare, reimpostare o ripristinare il PIN (una volta eseguita la disconnessione), visitare [Reimpostare il PIN una volta eseguita la disconnessione da Windows 10](https://support.microsoft.com/windows/reset-your-pin-when-you-aren-t-signed-in-to-windows-10-a386c519-3ab2-b873-1e9b-bb228a98b904).

- Per configurare il PIN di Windows Hello, visitare [Informazioni su Windows Hello e relativa configurazione](https://support.microsoft.com/windows/learn-about-windows-hello-and-set-it-up-dae28983-8242-bb2a-d3d1-87c9d265a5f0).
