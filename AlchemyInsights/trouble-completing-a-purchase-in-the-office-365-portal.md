# <a name="trouble-completing-a-purchase-in-the-microsoft-365-admin-center"></a>Non si riesce a completare un acquisto nell'interfaccia di amministrazione di Microsoft 365

Quando si prova a effettuare un acquisto, se è stato chiesto più volte di accedere all'interfaccia di amministrazione di Microsoft 365 o se un messaggio indica che si è verificato un problema, è consigliabile prima di tutto provare a usare la modalità di navigazione InPrivate o in incognito. Dopo aver aperto una finestra di navigazione InPrivate o in incognito, passare al sito [admin.microsoft.com](https://admin.microsoft.com) e accedere con le credenziali di amministratore. Quindi concludere l'acquisto.
