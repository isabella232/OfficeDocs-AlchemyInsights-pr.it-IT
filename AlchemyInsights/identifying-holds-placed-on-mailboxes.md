# <a name="identify-holds-placed-on-mailboxes"></a>Identificare i blocchi inseriti nelle cassette postali

Si dispone di Microsoft 365 contenuto che viene conservato **da** un blocco **che** non è possibile identificare? È necessario **scoprire dove** si trova il blocco sulla cassetta **postale** ? I blocchi (*funzionalità di* conservazione ) possono essere applicati tramite uno dei metodi seguenti:
  
- Conservazione in caso di dispute

- Blocco di eDiscovery

- Blocco sul posto

- Microsoft 365 di conservazione 

  - Criteri di conservazione percorso specifici

  - Criteri di conservazione a livello di organizzazione

- Microsoft 365 etichette

Per identificare il tipo di blocco applicato alla cassetta postale di un utente, vedere [How to identify the type of hold placed on an Exchange Online mailbox](https://docs.microsoft.com/microsoft-365/compliance/identify-a-hold-on-an-exchange-online-mailbox).
