# <a name="sharing-and-permission-methods-to-provide-or-restrict-user-access-to-sharepoint-site"></a>Metodi di condivisione e autorizzazione per fornire o limitare l'accesso degli utenti SharePoint sito

Dopo aver creato un sito di SharePoint, può essere opportuno fornire o limitare l'accesso degli utenti al sito o al relativo contenuto. Questa operazione può essere eseguita tramite vari metodi di condivisione e autorizzazione indicati di seguito.

- [Condividere file o cartelle di SharePoint](https://support.office.com/article/share-sharepoint-files-or-folders-1fe37332-0f9a-4719-970e-d2578da4941c)

- [Condividere un sito](https://support.office.com/article/share-a-site-958771a8-d041-4eb8-b51c-afea2eae3658)

- [Modificare il tipo di collegamento predefinito quando gli utenti ricevono collegamenti per la condivisione](https://docs.microsoft.com/sharepoint/change-default-sharing-link)

- [Interrompere la condivisione di file o cartelle di OneDrive o Share Point o modificare le autorizzazioni](https://support.office.com/article/stop-sharing-onedrive-or-sharepoint-files-or-folders-or-change-permissions-0a36470f-d7fe-40a0-bd74-0ac6c1e13323)

- [Condivisione e autorizzazioni](https://support.office.com/article/Sharing-and-permissions-ac85fbf1-2431-49bf-8690-f1a2b98af65f#ID0EAABAAA=Manage_permissions)

- [Attivare o disattivare la condivisione esterna per Sharepoint Online](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off)

- [Errori di condivisione OSE in Sharepoint e OneDrive](https://docs.microsoft.com/sharepoint/sharepoint-onedrive-error-message)
